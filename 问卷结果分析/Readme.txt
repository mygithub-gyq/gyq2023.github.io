1.描述性统计分析
频率占比，均值，标准差，结果解释
2.信效度检验
针对量表题，1-7衡量程度的不同
3.影响因素分析
提取维度数据，然后根据问题的选项个数，两个t检验，检验均值，多个类别，F检验，检验方差，最终给出结果解释
4.多元回归分析
提取自变量，因变量，计算出数据结果，然后逐步线性回归，选出最优自变量，得到回归结果，R方，VIF值，沃森值，p值，检验方程的显著性，系数的显著性，对结果进行解释
5.SEM结构方程模型
先将多个自变量分类，然后探究和因变量的关系，因变量也可以是多个，需要检验统计量，p值，模型路径，结果解释，更高级，可以将分组后的依次和因变量做路径分析，也可以一起做
结构方程模型（Structural Equation Modeling, SEM）是一种综合分析方法，可以用于探索 观测变量（Observed Variables） 与 潜变量（Latent Variables） 之间的关系。相比传统回归分析，SEM 可以： ✅ 同时处理多个因果关系✅ 包含观测变量和潜变量✅ 进行路径分析和因果推断
